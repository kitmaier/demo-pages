Is π^3.14 or 3.14^π bigger?

As a rule of thumb, changes to the exponent should have greater impact than changes to the base, so it should be the case that:

3.14^3.14 < π^3.14 < 3.14^π < π^π

We can rearrange the target expression as follows:

π^3.14 =? 3.14^π

3.14*ln(π) =? π*ln(3.14)

ln(π)/π =? ln(3.14)/3.14

This now represents the difference in output given two different inputs for the same function f(x) = ln(x)/x.

This function can be analyzed to determine its behavior.

ln(x) will be defined for x>0, zero at x=1, and increasing everywhere, quickly when negative and slowly when positive.

1/x will be defined for x>0, and positive and decreasing rapidly.

ln(x)/x then will be expected to have a single maximum value for some x>1, and be increasing up to that value and decreasing after it.

This unique maximum value can be determined with calculus.

d/dx(ln(x)) = 1/x

d/dx(ln(x)/x) = (1-ln(x))/x^2

(1-ln(x))/x^2 = 0

1-ln(x) = 0

ln(x) = 1

x = e

Because e < 3.14 < π it follows that f(3.14) > f(π).

ln(π)/π < ln(3.14)/3.14

3.14*ln(π) < π*ln(3.14)

π^3.14 < 3.14^π
